# 📙 About <a name = "about"></a>
Blog maker is a simple blog maker app built using Vue js.


# 🏁 Get started <a name = "install"></a>
```
git clone https://github.com/gaserashraf/Blog-Maker.git
```
```
cd Blog-Maker
```
```
npm install
```
```
npm run serve
```

# 💻 Built using <a name = "tech"></a>
- **Vue js**
